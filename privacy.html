墨韵练字 隐私政策

更新日期：2026年6月20日  |  生效日期：2026年6月20日

「墨韵练字」（以下简称「本应用」）由开发者提供，用于导入 TTF/OTF 字体并进行屏幕描红、对照练字。我们重视您的隐私，本政策说明我们如何收集、使用、存储及保护您的信息。

一、我们收集的信息

本应用不注册账号，不主动向开发者服务器上传姓名、手机号、身份证号、位置信息、设备唯一标识等个人身份信息。

在使用本应用时，可能涉及以下本地数据（仅保存在您的设备上）：

  • 内置字体：安装包内预置的开源书法字体（马善政、龙藏、志芒行），仅在应用内加载用于练字，开发者不会上传至自有服务器。
  • 字体文件：您通过系统文件选择器主动导入的 TTF/OTF 字体（需 VIP），仅在应用内加载用于显示与练字，开发者不会上传至自有服务器。
  • 练习内容：您在应用内输入或粘贴的文字，用于单字切换与描红练习，保存在设备本地。
  • 名篇库：您保存到「我的名篇」的文本，以 JSON 文件形式存储在应用私有目录。
  • 内置名篇：应用安装包内预置的古诗词、文言名篇，仅供本地阅读与练习。
  • VIP 状态与激活码：是否已开通 VIP、已兑换的激活码标识，均保存在设备本地，用于控制功能权限。

二、权限说明

  • 存储 / 文件访问：用于通过系统文件选择器读取您选择的字体文件。仅在您主动点击「导入字体」时使用，不会扫描或上传其他文件。
  • 网络（INTERNET）：在您主动使用 VIP 购买相关功能时使用，包括：（1）华为应用内支付（HMS IAP）；（2）通过系统浏览器打开爱发电（afdian.com）购买页面或收银台；（3）华为应用市场版本更新检查。除上述由您主动触发或进入主界面后的更新检查外，本应用不会后台联网上传您的练字内容或字体文件。

三、应用内购买与第三方支付

本应用提供 VIP 永久版，用于解锁自定义导入 TTF/OTF 字体。您可选择以下方式（均为您主动发起）：

  • 华为应用内支付：在已安装 HMS Core 的华为 / 荣耀设备上，通过华为应用内支付（HMS IAP）完成购买。支付流程由华为处理，开发者不接触您的支付密码或完整支付账号信息。
  • 爱发电（afdian.com）：在非华为 Android 设备或您选择备用方式时，本应用会打开系统浏览器跳转至爱发电商品页或收银台，您可使用支付宝、微信等完成支付。支付与爱发电账号相关数据处理遵循爱发电平台规则。
  • 激活码：您通过上述渠道获得激活码后，在本应用内输入兑换；激活码校验在设备本地完成，开发者不建立用户登录体系。

内置 3 款开源字体永久免费；自定义导入 TTF/OTF 字体需开通 VIP。VIP 为一次性永久解锁，非订阅制自动续费（华为 IAP 商品类型为非消耗型）。应用内「字体许可」页可查看内置字体开源授权说明。

四、信息的存储与删除

  • 练字相关用户数据均存储在您的设备本地，开发者无法远程访问。
  • 卸载本应用后，应用私有目录中的数据（含「我的名篇」、VIP 本地状态）将被系统删除。
  • 您可在应用内删除已保存的名篇，或清除已导入的字体。

五、第三方 SDK 与服务

  • Flutter 及开源插件（如 file_picker、path_provider、shared_preferences、url_launcher）：用于文件选择、本地存储、打开浏览器等，不向开发者自有服务器传输个人信息。
  • 华为 HMS 应用内支付 SDK（huawei_iap）：仅在 Android 华为 / 荣耀设备上、您主动购买 VIP 时调用，用于完成应用内支付与恢复购买。具体数据处理见华为隐私声明。
  • 华为应用市场更新（huawei_in_app_update）：在华为 / 荣耀设备进入主界面后，可能检查应用市场是否有新版本并提示更新。
  • 爱发电（afdian.com）：第三方赞助 / 商品平台，仅在您点击购买链接时使用，由浏览器访问，非应用内嵌 WebView 静默采集。

六、未成年人保护

本应用为练字工具，不面向未成年人收集个人信息。应用内购买需由监护人同意并在平台规则下完成。若您是未成年人的监护人并对本政策有疑问，请通过下方联系方式与我们沟通。

七、您的权利

依据相关法律法规，您享有以下权利：

  • 知情权：您有权了解本应用如何处理您的信息，本政策即为此说明。
  • 访问与更正：您可在应用内直接查看、编辑练习文字及「我的名篇」内容。
  • 删除权：您可删除名篇、清除字体，或通过卸载应用删除全部本地数据。
  • 撤回同意：您可随时停止使用本应用；字体导入、VIP 购买均需您主动操作。
